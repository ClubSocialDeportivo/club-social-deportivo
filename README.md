# Club Social Deportivo

Sistema de gestión de socios desarrollado con Laravel, React y PostgreSQL.

## Tecnologías utilizadas

Backend:
- Laravel
- PHP
- PostgreSQL
- API REST

Frontend:
- React
- JavaScript
- Vite

## Estructura del proyecto

backend/  
API desarrollada en Laravel que maneja la lógica del sistema y la conexión con la base de datos.

frontend/  
Aplicación desarrollada en React para la interfaz de usuario.
